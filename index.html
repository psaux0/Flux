Flux

Flux is an open-source functional shirt designed for body temperature regulating. It is generated at Open Style Lab in summer 2014 by a team made of a designer, an engineer and an occupational therapist. Currently Flux is still under development.

Features

  • Automatic temperature regulating
  • Remote temperature monitoring
  • Universal designed web application
  • Android App (under development)

Software Dependencies

Server Side (for web application)

Web server is written in Python. To communicate with microcontroller, we use PySerial. Web server application is built on light weight framework Flask.

Microcontroller

There are two versions of Flux right now. Flux 1.0 is based on Arduino Uno, so there won't be additional dependencies on microcontroller side. Flux 1.1 is based on ATtiny45/85. To program AVR microcontroller, AVR studio is necessary.

Android App

If you want to modify or contribute to the Android App, we would suggest you use the Eclipse IDE for Android. Android with version over 4.4 is needed for building this app.

PCB Design

All PCBs were designed in Eagle 6.4. In order to find all the components we used, you need install Adafruit and Sparfun eagle libraries.
